함수 >> 정의와 호출
함수에게 이름을 지어주고, 기능을 불어넣어준다.

함수 3가지 체크
1. 정의와 호출
함수에게 이름을 지어주고, 기능을 불어넣어준다.
2. 입력값(파라미터)
함수에게 던져주는 값. 입력값이 없는 함수도 만들 수 있다.
3. 리턴값(돌려주는 값, 반환값)
함수가 기능을 모두 마치고 돌려주는 값.
언어에 따라서 돌려주는 값의 자료형을 명시하는 경우가 있음

ex)

#include <stdio.h>

int add(int a, int b) {
	return a+b;
}

int main() {
	int result = add(3,5);
	printf("출력값 : %d\n" , result);
	return 0;
}

1. 정의와 호출
add라고 이름을 붙여줌 add(3,5)로 호출
2. 입력값(파라미터)
함수 내에서 a와 b로 쓰겟다고 정의.
3. 리턴값(돌려주는 값, 반환값)
함수에서는 a+b 를 돌려주고 있다.
c언어에서는 리턴값의 자료형은 함수명 앞에다가 적는다
여기서는 int로 되있으니 숫자형을 돌려준다는것

C언어 함수 
1. 함수 앞에 리턴값 자료형이 잇더라
2. main 이라고 적힌 함수부터 코드가 시작

Java 메서드
1. 메서드 앞에 뭔가 용어가 많음
2. c언어와 같이 리턴값에 대한 자료형이 있다.
3. c언어와 같이 main 함수부터 코드가 시작

Python 함수
1. 함수를 만들때 def 라는 키워드를 쓴다
2. 자료형에 대한 힌트가 없다.
3. 함수 내용을 괄호가 아닌 "들여쓰기" 로 구분
