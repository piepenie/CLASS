비교 연산자

c언어는 조건문에 0이 나오면 거짓
1이 나오면 참이된다.
추가로 1이 아닌 다른 숫자를 참으로 판단함
if(2) => 참입니다
if() << 안에 0이 아닌 다른 숫자가 들어가면 모두 참임

조건문에 문자가 들어가면?
문자 => 참
빈문자 => 거짓
역슬레쉬(\) => 거짓(역슬레쉬가 들어가기만해도 거짓)

for 초기식과 조건식을 잘봐야한다
while 조건식과 break를 잘봐야한다
do.. while 은 처음에 무조건 한번 실행되는 부분을 봐야한다

초기식 : 반복문이 처음 시작할 때 단 한번 실행됨
조건식 : 이 식이 참인 동안 반복문 실행, 조건식이 거짓이되면 반복문이 종료

증감식 : 반복문이 한번 실행될 때마다 조건식을 검사하기 전에
실행됨
(1. 초기식 > 2. 조건식 >> 3.코드 실행 >> 4.증감식 >> 5.조건식)

while 은 조건이 참인 경우만 계속 반복
조건문이 종료되는 순간을 잘 봐야함.

do..while 형태
do..while 반복문은 조건을 검사하기 전에 코드를 먼저 실행
반복 블록을 최소 한 번 실행한 후 조건을 검사하여
반복을 계속할지 결정

일단 do 블록 안에 있는 것을 최초로 실행된다는 것을
잊지말자.

C언어 : 조건식에 0이 되는 경우에는 거짓으로 인식,
0이 아닌 모든값이 있으면 참으로 인식

Java : 조건식에 단순 숫자가 들어가면 오류가 발생.
명확하게 들어가야하며, 트루 폴스 단어로 가능
(boolean 타입)

Python : 파이썬에서 비어 있지않은 객체(리스트 문자열 튜플)
은 참으로 판단하고, 0이나 비어있는 객체의 경우 거짓으로 판단

&& >> 앞뒤가 모두 참으로 인식되어야 전체를 참으로 인식
|| >> 앞 뒤 둘중 하나만 참이면 전체가 참
