변수

데이터를 저장할 수 있는 메모리의 기억 공간

변수 생성 규칙 및 특징
1. 예약어(키워드)는 사용할 수 없다.
2. 공백을 포함할 수 없다
3. 첫 글자는 영문과 언더버(_)만 사용이 가능하다
4. 언더바(_) 이외의 특수문자는 사용할 수 없다.
5. 대소문자를 구분한다.

데이터 타입(자료형)
bit => 0과1 두가지 경우만 넣을 수 있음
1Byte = 8bit (8개의 상자를 가지고있음) => 1바이트의 경우의 수 = 2^8 => 256가지.

문자형 char 1Byte
정수형 short 2Byte
         int    4Byte
         long  4Byte
실수형 float  4Byte
         double 8Byte
