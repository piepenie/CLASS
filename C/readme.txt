C언어 프로그래밍이란,

프로그램 동작 과정
선언 >> 입력 >> 처리 >> 출력

ex)
선언 : 제목과 내용을 쓸 수 있는 곳을 만듬
입력 : 제목이나 내용을 적음
처리 : 엔터를 눌러서 글을 저장함
출력 : 저장된 제목과 내용을 보여줌

상식적인 내용을 컴퓨터가 이해할 수 있게 해주는것
>> 프로그래밍

ex) 아침에 비가 오면 우산을 가져가고
비가 오지 않으면 자전거를 타고간다.

#include <stdio.h>
#include <string.h> // strcmp 함수를 사용하기 위해 필요

int main() {
    char 자전거[] = "자전거";  // 문자열 저장
    char 우산[] = "우산";      // 문자열 저장
    char 날씨[10];            // 사용자 입력을 받을 문자열 배열

    printf("날씨를 입력하세요: ");
    scanf("%s", 날씨);  // 사용자로부터 날씨 입력 받기

    if (strcmp(날씨, "비") == 0) {  // 문자열 비교
        printf("%s\n", 우산);
    } else {
        printf("%s\n", 자전거);
    }

    return 0;
}
